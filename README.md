# Bot Cambridge One - Locaso Cambridch 🤖🚀

Este bot automatiza la resolución de ejercicios en la plataforma **Cambridge One**. 

## ✨ Características
- **Resolución Automática**: Detecta las respuestas correctas y las ingresa por ti.
- **Soporte Multi-Pantalla**: Navega automáticamente entre las diapositivas del ejercicio.
- **Multi-Ejercicio**: Al terminar una actividad, salta automáticamente a la siguiente ("Next activity").
- **Tipos de Ejercicio Soportados**: 
  - Drag & Drop (Word Bank)
  - Dropdowns (Selects)
  - Multiple Choice
  - Text Entry

## 🚀 Cómo usar

### 1. Requisitos
- Tener **Brave Browser** instalado.
- Python 3.x instalado.
- Instalar dependencias:
  ```bash
  pip install -r requirements.txt
  ```

### 2. Configuración
1. Cierra todas las ventanas de Brave.
2. Ejecuta el archivo `lanzar_brave_debug.bat`. Esto abrirá Brave en modo depuración.
3. Inicia sesión en Cambridge One y ve al ejercicio que quieras resolver.

### 3. Ejecución
Corre el script:
```bash
python script.py
```
Presiona **ENTER** en la consola y el bot hará el resto.
